# 竞品技术分析自动化脚本
* author:zhuhuanhuan
* time:2017/08/02
* version:1.0.0

## 技术说明：
* groovy使用
* gradle管理依赖

## 使用说明
* SOURCE_DIR：放置apk包的地方
* OUTPUT_DIR：放置反编译出来包的地方
* PARSE_DIR：放置分析文件的地方
* Main.groovy文件中的SOURCE_DIR，OUTPUT_DIR，PARSE_DIR设置成你电脑的路径
* gradle run执行  或者 鼠标放在Main.groovy内，右击“run main”就行了
